The Learning Journey: From Assignment 1 to Assignment 5

Project Overview

This repository reflects my learning progress from Assignment 1 to Assignment 5, showcasing the development of my skills in data cleaning, merging datasets, and applying what I have learned across multiple assignments. Below is a breakdown of each assignment and how it contributed to my learning:

Assignment 1: Introduction to GitHub and Jupyter Notebooks

In Assignment 1, I was introduced to GitHub and Jupyter notebooks:

  • GitHub: I learned how to use GitHub for version control, including how to create repositories, push and pull changes, and manage branches.
  • Jupyter Notebooks: I became familiar with Jupyter notebooks as an environment for interactive coding and documentation.

This assignment laid the foundation for managing code and data, and was my first experience with using GitHub for a project.

Assignment 4: Automating Data Cleaning with Jupyter Notebook

In Assignment 4, I used Jupyter notebooks to automate the cleaning of a student survey dataset:

  • I explored the dataset to identify missing values and outliers.
  • I applied data cleaning techniques such as handling missing values, filtering out irrelevant data, and standardizing formats.
  • The notebook allowed me to automate the cleaning process, which made it easier to handle large datasets in a reproducible manner.

This assignment helped me develop a deeper understanding of data preprocessing and how to automate tasks using Python in Jupyter notebooks.

Assignment 5: Merging Datasets and Handling Missing Data

In Assignment 5, I applied the skills from Assignment 4 to merge the cleaned student survey dataset from Assignment 4 with a new fake dataset:

  • The fake dataset, 2024 salary by age group, was generated to simulate salary data for different age groups.
  • I merged these two datasets to create a comprehensive dataset that could be used for analysis.
  • I documented the merging process and handled NA values that were created during the merge.

This assignment allowed me to demonstrate my ability to merge multiple datasets and manage missing data, which is a critical skill in data analysis.

Conclusion

Throughout these assignments, I have built upon my knowledge and skills:

  • Assignment 1 introduced me to GitHub and Jupyter notebooks, which are essential tools for managing projects and coding.
  • Assignment 4 taught me how to clean and preprocess data using Python and Jupyter notebooks.
  • Assignment 5 gave me the opportunity to apply data cleaning techniques to merge datasets and handle real-world issues like missing data.

This progression has helped me develop a strong foundation in data management and analysis that will be valuable for future projects.

Project Structure

The project contains the following files and folders:

  • BABI4005Assign1.md: The first assignment documenting the initial project setup.
  • Data Folder:
    • 2024_salarybyagegroup_data.csv: Fake dataset generated to merge with the cleaned survey dataset.
    • clean_survey_2025-02-06_assign4_sosochoi.csv: Cleaned dataset from Assignment 4.
  • Clean_Data Folder: Stores the cleaned dataset (clean_survey_2025-02-06_assign4_sosochoi.csv).
  • Scripts Folder:
    • gen_fake_data.ipynb: Generates the fake salary dataset.
    • Assign5_Merge_dataset.ipynb: Merges the cleaned survey dataset with the fake dataset and explains the merge process.
  • Documents Folder: Contains a Word document with a screenshot of the local version of the project.
  • Results Folder: Contains the output file merged_student_data.csv, generated after executing the Assign5_Merge_dataset.ipynb script.
