The Learning Journey: From Assignment 1 to Assignment 5

Project Overview

This repository reflects my learning progress, starting from Assignment 1 to the final project (Assignment 5). The goal is to showcase the development of skills in data cleaning, merging datasets, and applying what I have learned across multiple assignments.

In this project, I have merged a cleaned dataset with a new fake dataset and documented the merge process, including any NA values created.

Assignment 1: Getting Started

In Assignment 1 (BABI4005Assign1.md), I focused on:

  • Learning the basics of GitHub (push, pull, and other fundamental concepts).
  • Getting familiar with Jupyter notebooks and their functionality.

Assignment 4: Cleaning the Data

In Assignment 4, I learned how to clean and preprocess datasets, focusing on:

  • Identifying and handling missing data.
  • Standardizing data formats.
  • Ensuring the dataset was ready for analysis.

Assignment 5: Merging the Datasets

In Assignment 5, I applied the skills learned in previous assignments to:

  • Merge the cleaned dataset with a fake salary dataset.
  • Document the merge process and the NA values created during the merge.
  • Apply data cleaning, dataset merging, and handling missing data concepts.

Final Thoughts on the Learning Progress

Each assignment built upon the skills I learned in the previous ones:

  • Assignment 1 helped me get familiar with GitHub and Jupyter notebooks.
  • Assignment 4 allowed me to clean and preprocess data effectively.
  • Assignment 5 demonstrated my ability to merge datasets and handle missing data (NA values) introduced during the merge.
